Journey

Short Description:
A web application for planning trips, viewing events, and managing itineraries with a user-friendly interface.

Features:

View all events with details.

Filter events

Create personalized trip plans.



Pages Description:

index.html / Homepage: The main page that provides an overview and navigation to all sections. Users can access events, the planner, and other main pages.

events.html: Displays a list of all events. Users can browse and filter events.

event-details.html: Shows full details of a selected event, including its name, date, location, and description. It also allows users to make payments and complete their reservations.

planner.html: Allows users to create, view, and edit their trip plans.

reservations.html: Displays a userâ€™s confirmed and pending reservations. Users can view, complete, or delete their reservations.

login.html: Handles user authentication. Users must log in to access protected pages.

sign-up.html: Allows new users to register for an account.

admin.html: Allows the admin to manage events.
